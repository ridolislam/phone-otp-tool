# 📱 Phone Number Bulk OTP Tool

এটি একটি অটোমেটিক টুল যা `numbers.txt` ফাইল থেকে ফোন নম্বর লিস্ট পড়ে, প্রতিটি নম্বরের জন্য ChatGPT সাইন-আপ ফর্মে "Send OTP" বাটন প্রেস করে।

## ✨ ফিচারসমূহ

- ✅ HTTP প্রটোকল ভিত্তিক (ব্রাউজার দরকার নেই)
- ✅ ব্যাচ প্রসেসিং (একসাথে অনেক নম্বর)
- ✅ অটোমেটিক রেট লিমিট হ্যান্ডলিং
- ✅ সফল/ব্যর্থ লগ সংরক্ষণ
- ✅ JSON ফরম্যাটে প্রোগ্রেস ট্র্যাকিং

## 📋 প্রয়োজনীয়তা

- Python 3.8 বা তার উপরে
- ইন্টারনেট সংযোগ

## 🚀 ইনস্টলেশন

```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/yourusername/phone-otp-tool.git
cd phone-otp-tool

# ডিপেন্ডেন্সি ইনস্টল করুন
pip install -r requirements.txt
